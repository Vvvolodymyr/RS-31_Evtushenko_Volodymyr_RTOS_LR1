# RS-31_Evtushenko_Volodymyr_RTOS_LR1
Номер групи: РС-31

Автор: Євтушенко Володимир

ОПЕРАЦІЙНІ СИСТЕМИ РЕАЛЬНОГО ЧАСУ ТА 32 БІТНІ МК Лабораторна робота 1. Технологія програмування портів вводу-виводу загального призначення на основі прототипу операційної системи нескінченного циклу опитування

Обладнання: STM32F411, CubeIDE.

Мета: Ознайомитися з принципами програмування GPIO STM32F411 через регістрові структури, реалізувати нескінченний цикл опитування кнопки та керування світлодіодами

Завдання до лабораторної роботи:

Варіант 5. Запрограмувати формування імпульсного сигналу на PD12 та PD13 з частотою 10 Гц, тривалість імпульсу 20 мс.

Функціонал:
LED Pulse Task: Генерує імпульсний сигнал тривалістю 20 мс на світлодіодах PB12 та PB15 з частотою 10 Гц.

Підключення:
LED1: GND → LED → резистор 220 Ом → PB12

LED2: GND → LED → резистор 220 Ом → PB15
